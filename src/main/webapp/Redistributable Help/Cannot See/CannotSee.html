Can't see the chart?

A template for you to customize...

It may be that the iXv Viewer in which the chart should appear fails to load, or that the iXv Viewer loads but the chart is incomplete. Alternatively, the iXv Viewer and chart may appear correctly, but an error occurs when you attempt to communicate with the server – perhaps when you search for information, or expand an entity. If any of these applies to you, check the following.

Are you using a personal firewall?

If you have a personal firewall installed on your computer, then the iXv Viewer may not respond when you click the Search button. To make the iXv Viewer work with a personal firewall, you must change the settings to "allow ads" or similar.

Do you have a Java Runtime Environment installed?

The iXv Viewer requires Oracle's Java Runtime Environment (JRE) to be installed on your computer.

If you see a "broken link" icon in place of the iXv Viewer, or a dialog requesting that you install Java, you can download a JRE from the Oracle website.

Are you sure you have a supported web browser?

  • Internet Explorer 8.0

  • Internet Explorer 7.0

  • Internet Explorer 6.0

  • Firefox 3.6

  • Firefox 3.5

  • Firefox 3.0

Are you sure you have a supported operating system?

  • Windows 7

  • Windows Vista Business SP2

  • Windows XP Professional x86 SP3

Does your computer meet the following minimum requirements?

  • 500 MHz processor

  • 512 MB RAM

  • 120 MB free hard disk space

  • 1024 x 768 high color (16-bit) display

Are you authorized to see the chart?

It is possible that you are not authorized to see the chart. For information on what you can and cannot see, please contact your data provider.

If you still can't see the chart...

Please contact your data provider for support.
