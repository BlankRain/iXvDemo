Search box

Enter criteria to search the data source for the information you want to find.

Some data providers may implement the search facilities on a separate web page.

Print on Single Page toolbar button

Prints the chart, scaled to fit on a single page.

Print at 100% Scale toolbar button

Prints the chart at full size, across multiple pages.

Page Setup toolbar button

Displays the Java Page Setup dialog, allowing the user to configure printing options.

Delete Selection toolbar button

Deletes selected entities, and any connecting links.

Undo Delete toolbar button

Returns the last set of entities you deleted to the chart.

Key Entity Emphasis toolbar button

Turns on and off the emphasis of entities that are highly connected to other entities on the chart.

Entities are emphasized by increasing the size of their image on the chart.

Time Bar toolbar button

Turns on the time bar and rearranges the chart as a timeline chart so that the chronology of timed events is shown.

Click the toolbar button again to turn off the time bar and rearrange the chart as an association chart.

Panning Tool toolbar button

Enables the panning tool, which allows you to move a whole chart intuitively in the iXv Viewer.

You can also double-click an item to center it in the iXv Viewer.

Zoom to Area toolbar button

Zooms in to a particular area on the chart.

Use the mouse to draw a box around the items you want to zoom in on.

Fit to Window toolbar button

Snaps the chart to the size of the iXv Viewer window.

Note that if you have many items, this may make them very small.

Fit Selection to Window toolbar button

Zooms in on just the currently selected items.

Select All toolbar button

Selects all the items on the chart.

Reorganize toolbar button

Reorganizes all the entities on the chart so you get an alternative snapshot of the information.

Find toolbar button

Opens the Find dialog box, where you can search for text on the chart.

Matching items are selected.

More... toolbar button

Appears when there are more buttons in the toolbar than can fit in the display. Opens a sub-toolbar containing the hidden buttons.

Animated logo

The logo animates while the iXv Viewer is processing your instructions or waiting for information.

Filter buttons

Click a filter button to prevent entities of that type from being displayed, making the chart easier to understand.

Click the same button again to restore them.
