Step 2 – Exploring the data source

To explore the data source, right-click an entity to open the shortcut menu. If the data provider has implemented expand functionality, select the appropriate menu command (for example, Show Relationships or Expand).

When you expand an entity, any related entities stored in the data source are visualized on the chart, and linked back to the expanded entity. The link may tell you the nature of the relationship.
