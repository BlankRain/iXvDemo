Delete Selection toolbar button

Deletes selected entities, and any connecting links.

Undo Delete toolbar button

Returns the last set of entities you deleted to the chart.

Panning Tool toolbar button

Enables the panning tool, which allows you to move a whole chart intuitively in the iXv Viewer.

You can also double-click an item to center it in the iXv Viewer.

Zoom to Area toolbar button

Zooms in to a particular area on the chart.

Use the mouse to draw a box around the items you want to zoom in on.

Fit to Window toolbar button

Snaps the chart to the size of the iXv Viewer window.

Note that if you have many items, this may make them very small.

Fit Selection to Window toolbar button

Zooms in on just the currently selected items.

Select All toolbar button

Selects all the items on the chart.

Reorganize toolbar button

Reorganizes all the entities on the chart so you get an alternative snapshot of the information.

Find toolbar button

Opens the Find dialog box, where you can search for text on the chart.

Matching items are selected.

Help toolbar button

Opens the iXv Viewer online help (for example, this help).

Zoom scrollbar

Zooms in and out of the chart.

Step 3 – Working with the chart

The iXv Viewer toolbar enables you to work with the entities and links on the chart. You can summarize the intelligence the chart contains, or maximize its impact upon third parties.

In addition, the data provider may implement a feature of the iXv Viewer that enables you to filter information from the chart temporarily, in order to focus on only the most important information.
