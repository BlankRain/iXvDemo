Step 5 – Transferring information to desktop applications

The iXv Viewer offers two ways to transfer information to desktop applications. The first, which is always available, uses regular drag-and-drop:

  1. Position or resize the iXv Viewer and the desktop application so that both windows are visible on the screen, with the iXv Viewer on top.

  2. In the iXv Viewer, select the items you want to transfer:

    • To select the entire chart, click the Select All toolbar button.

    • To select part of a large chart, hold down the left mouse button and draw a box from top left to bottom right around a group of items.

    • To select just a few items, hold down the Shift key while you click on those items.

  3. Holding down the left mouse button, drag the selected items out of the iXv Viewer and into the desktop application.

  4. When the cursor changes to look like this:

    release the mouse button. The selected items are visualized in the desktop application.

The second way to transfer information – when the iXv Viewer has been configured to do so – is easier still:

  1. In the iXv Viewer, select the items you want to transfer.

  2. Click the Send To Notebook toolbar button.

    If Analyst's Notebook is not already running, clicking the button starts it up. Then, the selected items are transferred to the desktop application.
